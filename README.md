# Basic Ruby Projects

## Description
These are some classic “intro to programming projects” that I made for learning 
purposes and The Odin Project.

## What is here
There are a total of four projects:
- Caesar cipher - (complete)
- Substrings - (complete)
- Stock picker - (WIP)
- Bubble sort - (WIP)

## Caesar Cipher
[Caesar Cipher project from The Odin Project](https://www.theodinproject.com/lessons/ruby-caesar-cipher) <br>
This program preforms a right shift Caesar cipher on the entered sentence,
 it is shifted by the number entered, then it outputs the encoded
sentence in the console.

## Substrings
[Sub-string project from The Odin Project](https://www.theodinproject.com/lessons/ruby-sub-strings) <br>
This program takes pre-entered text and a pre-entered dictionary and then
tallies up all of the times the words in the dictionary appear anywhere in the
text if word in the dictionary appears in the text. <br>
There are two texts in this program, a short text and a long text.
